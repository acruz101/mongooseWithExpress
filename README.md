**Farm Stand Application using MongoDB, Express Node.js, and Bootstrap**
- Project for The UDEMY Web Developer Bootcamp 2022 with Colt Steele
- Purpose of Project:

  - To practice Dynamic HTML templating with EJS partials and Bootstrap
